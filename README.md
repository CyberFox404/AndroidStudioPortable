# AndroidStudioPortable

Форк **CrazyMax** https://github.com/portapps/android-studio-portable


### Описание

Сборщик портативной версии из установочного файла **Android Studio**.

### Как пользоваться
1. Скачиваем установочный файл [**Android Studio**](https://developer.android.com/studio) с официального сайта (далее **AS**).
2. Скачивает архив **AndroidStudioPortable**.
3. Распаковываем архив **AndroidStudioPortable** (далее **ASP**).
4. Перетаскиваем скаченный установочный файл **AS** и кидаем на файл **Unpack.bat** находящийся в **ASP**.
5. Ожидаем завершения операции.
6. Будет создана директория **AndroidStudioPortable** внутри **ASP**, это готовая портативная версия **Android Studio**.
